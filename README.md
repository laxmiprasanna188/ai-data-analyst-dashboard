# AI Data Analyst Dashboard

## 📌 Project Overview
This project is an AI-powered data analyst dashboard that allows users to upload CSV files and automatically perform data cleaning, analysis, and visualization.

## 🚀 Features
- Upload CSV dataset
- Automatic data cleaning
- Data visualization (charts & graphs)
- Generate insights from data

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib / Plotly
- Streamlit

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## 📊 Sample Use Case
Analyze sales data to find trends and insights.
